# Sistem Pendataan Inventaris BPS Kabupaten Boyolali

Aplikasi web manajemen inventaris laptop untuk BPS Boyolali. Memudahkan pencatatan peminjaman, pengembalian, dan monitoring stok secara real-time.

**Fitur:**
*   Dashboard Ketersediaan Unit
*   Input Peminjaman & Pengembalian
*   Riwayat Transaksi & Filter
*   Integrasi Google Sheets (GAS)

**Teknologi:** HTML5, CSS3, JavaScript, Google Apps Script.
